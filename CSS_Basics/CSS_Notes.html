Nested tag is something that is nested in another tag. Some are required to be Nested, others should not be nested.

An Attribute is further instructions for the computer given inside of an opening tag. Such as the id we gave our div OR the charset we use inside of the metatag.

Lists are just like a list that you would make in real life. They require li tags inside of other tags.
